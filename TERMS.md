## 2. Description of Service

Piano Practice Pal is a free educational application designed to help users learn and practice piano scales. The App provides:
- 48 major, natural minor, harmonic minor, and melodic minor scales
- Audio playback using Web Audio API
- Visual piano keyboard and music staff notation
- Practice tracking and spaced repetition scheduling
- Offline functionality
