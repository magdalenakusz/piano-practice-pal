## Web APIs

This application uses browser-provided Web APIs:
- **Web Audio API** - For audio synthesis (no external library)
- **LocalStorage API** - For data persistence (browser native)
- **Service Worker API** - For offline functionality (browser native)

These APIs are provided by web browsers and are not subject to additional licensing.
